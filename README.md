# 🎮 Python Game Collection (Python 游戏合集)

一个包含多款经典游戏的 Python 命令行游戏合集。本项目集成了**贪吃蛇**、**井字棋**和**猜数字**三款游戏，旨在展示 Python 在控制台交互、逻辑算法及游戏状态管理方面的应用。

> **当前版本**: v2.0  
> **适用平台**: Windows (主要), macOS/Linux (部分游戏需调整)

---

## 🕹️ 游戏列表

### 1. 🐍 贪吃蛇 (Snake)
经典的贪吃蛇游戏，完全在命令行中渲染。
- **特色**:
  - 支持 Emoji 图形化显示 (🐍, 🟩, 🍎)。
  - 流畅的键盘控制 (方向键 或 WASD)。
  - 实时碰撞检测 (撞墙/咬自己)。
  - 自动适配 PyInstaller 打包后的资源路径。
- **操作**: 使用方向键或 `W/A/S/D` 控制移动，`Q` 退出。

### 2. ⭕ 井字棋 (Tic-Tac-Toe)
智能版井字棋，支持人机对战和双人对战。
- **特色**:
  - **双人对战 (PvP)**: 适合朋友面对面切磋。
  - **人机对战 (PvE)**:
    - 🟢 **简单模式**: 电脑随机落子，适合新手。
    - 🔴 **困难模式**: 基于 **Minimax 算法**，电脑永不失误（你能逼平它吗？）。
  - 清晰的控制台棋盘绘制与位置指引。
- **操作**: 输入数字 `1-9` 对应棋盘位置。

### 3. 🔢 猜数字 (Guess Number)
源自 `main.py` 的经典逻辑游戏。
- **特色**:
  - **人来猜**: 挑战你的直觉与二分法逻辑，系统会评估你的猜测次数。
  - **AI 来猜**: 见证算法的力量，AI 通过二分查找法快速猜出你心中的数字。
  - 智能反馈与评价系统。
